Tanagra Mapping Tool

Ce projet vise à mettre en place une interface web pour la géolocalisation et la cartographie des noms de lieux dans les textes. À partir d’un corpus numérisé, l’application ciblée permet d’identifier les noms de lieu à l’aide d’un outil déjà développé. Ensuite, les lieux reconnus sont géolocalisés en utilisant une base de données ouverte, puis projetés sur une carte interactive.
